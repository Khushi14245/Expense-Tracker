# EXPENSE TRACKER

A simple Python-based Expense Tracker that allows users to record their expenses, categorize them, save them to a CSV file, and view a summary of their spending along with their remaining budget.

## It includes the following features:
- Add a new expense
- Categorize expenses into:
  - Food
  - Home
  - Work
  - Fun
  - Miscellaneous
- Store expenses in a CSV file
- View total spending by category
- View total expenses
- Calculate the remaining budget
- Display the remaining days in the current month
- Calculate the daily budget based on the remaining budget

## The technologies used here are:
- Python
- Object-Oriented Programming (OOP)
- File Handling
- Dictionaries
- `datetime` module
- `calendar` module
